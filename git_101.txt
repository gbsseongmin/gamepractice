An understading for git: git basics and 101s.

1. Commands
    git add [file/folder]
adds the specified files and folders to the 'modified' state.

    git commit (-m, ...)
submits the changes to the local branch. Usually needs a message with '-m' command.
i.e; git commit -m "this is my first commit."
commits the modified changes with message "this is my first commit."
Check below to acknowledge about the message rules.

    git push [branch]

    git pull [branch]
